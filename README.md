# LLM Systems & Applied Generative AI

## Project Overview

This project focuses on building practical applications using Large Language Models (LLMs) and Generative AI technologies. The goal is to explore modern AI system design, Retrieval-Augmented Generation (RAG), prompt engineering, vector databases, and AI-powered assistants that can solve real-world problems.

The project serves as a hands-on implementation of concepts learned during my internship in LLM Systems & Applied GenAI.
---

## Objectives
Understand how Large Language Models work.
Build AI-powered applications using modern LLM frameworks.
Implement Retrieval-Augmented Generation (RAG).
Explore vector databases and semantic search.
Design scalable AI system architectures.
Evaluate and optimize LLM responses.
Apply prompt engineering techniques.
Develop production-ready GenAI workflows.
